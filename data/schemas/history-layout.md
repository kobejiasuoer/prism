# History Layout

This directory stores real historical Prism outputs after secret/privacy scrub.

- `ai_history/` contains AI screening snapshots.
- `quality_gates/` contains quality-gate snapshots.
- `cron_logs/` contains workflow run logs.
- `stale_outputs/` contains previous outputs retained for auditability.
- `reports/` contains Markdown and text report artifacts.
- `command_brief/` contains control-panel generated daily command briefs.
- `control_panel_runs/` contains control-panel run metadata and logs after privacy scrub.
- `daily_snapshots/` contains watchlist snapshot inputs referenced by command briefs.
